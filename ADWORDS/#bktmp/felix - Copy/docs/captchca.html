The captchca class provides an extra level of data entry security. The captchca requires storage in a session variable, and does not load session variables. You will need to start a session separately if you wish to use captchca.

Captchca

Captchca Caller

app/calls/captchca_call.php

  • Creates instance of class -> $captchca
  • Creates random 5 char string for captchca
  • Creates and saves image in filesystem for captchca
  • Returns captchca image string with img tags -> $captchca_image_show
  • Adds full filesystem path for captchca image to: $_SESSION['captchca_image']
  • Adds captchca text to: $_SESSION['captchca_text']

Captchca Class

app/captchca/captchca.inc.php

public function generate_filename()

  • Creates and returns random filename for captchca -> xxxxx.jpg

public function generate_random()

  • Creates and returns random string for captchca

public function generate_captchca_images($image_loc, $string, $save_loc)

  • Creates and saves image for captchca
  • $image_loc -> String -> file system location of the raw source image.
  • $string -> String -> random 5 char string produced earlier by generate_random().
  • $save_loc -> String -> file system save location for image.

public function show_captchca_image($image_loc)

  • Produces image html with img tags to display captchca image.
  • $image_loc -> String -> http location of the raw source image.
