app/imaging/imaging.inc.php

public function add_watermark($main_image, $watermark, $target_file, $image_quality, $transparency)

Sets a watermark based on watermark input file. Saves new file to desired location. Must use PNG 8 for background transparency to work.

$main_image -> path and filename of source image
$watermark -> path and filename of png8 watermark image
$target_file -> path and filename of the location to save the file
$image_quality -> passed from config - quality of final image 0-100
$transparency -> passed from config - transparency of final image 0-100

public function check_and_resize_image($file, $target_dir, $desired_width, $desired_height, $image_quality)

Resizes and saves image to desired location.

$file -> path and filename of source image
$target_dir -> path and filename of the location to save the file
$desired_width -> desired final width. If set to false, will use desired height and resize image to same aspect ratio as original.
$desired_height -> desired final height. If set to false, will use desired width and resize image to same aspect ratio as original.
Either $desired_width or $desired_height or both MUST be set. If both are set to false, nothing will happen.
$image_quality -> passed from config - quality of final image 0-100
