Database Classes

app/dbase/dbase_setters.inc.php

public function insert_query($table, array $cols)

Inserts values into database - returns nothing.
$table = string -> name of database.
$cols = multidimensional array of column names then entries -> [$i][0] = col name [$i][1] = entry

public function update_query($table, array $cols, array $where = null)

Updates data into the database - returns nothing.
$table = String -> table name
$cols = multidimensional array of column names then entries -> [$i][0] = col name [$i][1] = entry
$where = (optional) multidimensional array of column names then entries -> [$i][0] = col name [$i][1] = entry

app/dbase/dbase_getters.inc.php

public function basic_get($table, $select_cols, $where_cols, $order, $limit)

Gets data from database - simple query, no joins. Returns array of mysql data.
$table = string name of table
$select_cols = single dimension array of column names to select. If false then select all
$where_cols = multi dimensional array of column names and values to select. [$i][0] = column name, [$i][1] = value . if false system leaves query blank
$order = multi dimension array of column names and asc or desc type [$i][0] = col name [$i][1] = asc / desc (represented by 0 or 1 respectively)
$limit = single dimension associative array -> ['lower'] = start num ['upper'] = # of public function get_last_id($table)elements to return

public function get_last_id($table)

Gets last entered id from database. Returns integer of last entered id.
$table = string name of table
