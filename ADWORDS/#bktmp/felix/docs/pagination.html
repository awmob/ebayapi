Pagination Classes

Pagination classes need to be called in the following order they appear here to work properly.

app/pagination/pagination.inc.php

public function paginate_numbers($count_of_entries, $items_per_page, $current_page, $pages_to_show)

  • Function gets the basic pagination numbers and sets static variables for use in next function
  • $count_of_entries = number of returned values from dbase
  • $items_per_page = number of desired items to appear on each page
  • $current_page = number of current page
  • $pages_to_show = number of pages to show in results

public function pagination_numbers_set()

  • Function sets the page numbers with next, prev, first, and last. Sets into static variables

public function set_pagination_html($prefix, $suffix)

  • Function sets the pagination html with chosen prefix (ie. Page) and suffix (ie. .htm) to for example Page1.htm
  • Returns associative array: ['prev'], ['first'], ['middle'][$x], ['last'], ['next']
  • ['prev'] -> Previous link. [0] = Single number value. [1] = suffix and prefix string. If null then no previous.
  • ['first'] -> First number. [0] = Single number value. [1] = suffix and prefix string. If null then no number.
  • ['middle'] -> Middle numbers. [$i][0] = single number value. [$i][1] = suffix and prefix string.
  • ['last'] -> Last number. [0] = Single number value. [1] = suffix and prefix string. If null then no number.
  • ['next'] -> Next link. [0] = Single number value. [1] = suffix and prefix string. If null then no next.
