Input Helpers

Main Caller

app/calls/main_caller.php

Automatically loaded at runtime
Creates instance of class -> $input_helper
Also creates instances of other helper classes.

Helper Config (shared config file between classes)

config/imaging/helper_config.php

$input_helper_settings['min_pass_length'] -> Minimum char length of password
$input_helper_settings['max_pass_length'] -> Max char length of password
$input_helper_settings['len_pass_error_msg'] -> Shows error message if length violated

$input_helper_settings['min_user_name_length'] -> Min char length of username
$input_helper_settings['max_user_name_length'] -> Max char length of username
$input_helper_settings['len_user_name_error_msg'] -> Shows error message if length violated

$input_helper_settings['main_user_name_char_regex'] -> Regex of allowed characters in username
$input_helper_settings['char_user_name_error_msg'] -> Shows error message if illegal characters used.

$input_helper_settings['alpha_space'] -> Regex allowing only alpha and spaces
$input_helper_settings['alpha_space_error_msg'] -> Shows error message if illegal characters used.

$input_helper_settings['password_salt'] -> Hashed salt for password security
$input_helper_settings['password_addenda'] -> Extra chars (non alpha numeric) for password security
$input_helper_settings['hash_algorithm'] -> string of password algorithm - default = "sha512"

$input_helper_settings['filename_sanitize'] -> Regex cleaning up the filenames for upload

$ascii_table[$i]['base'] -> The basic special character
$ascii_table[$i]['code'] -> The ascii code for the special character

Input Helper Class

app/helpers/input_helper.inc.php

function __construct($input_helper_settings)

Sets the class and passes the variables from config file for use as private variables in the class.

public function set_hash($string)

Converts the string to desired hash algorithm - esp for passwords. Uses password salt in config prior to hashing for further security.

Returns: hashed string

$string = String to be converted

public function check_new_pass($pass)

Checks if password is of the required length.

Returns: associative array['error_exists'] = (true or false) -> $error['error_message'] = string with error message.

$pass = Password to be checked. String.

public function check_new_username($username)

Checks if username is of the required length and doesn't contain illegal characters.

Returns: associative array['error_exists'] = (true or false) -> $error['error_message'] = string with error message.

$username = Username to be checked. String.

public function set_unset_ascii_text($string, $to_ascii = true)

Replaces instances of ascii text with normal, and vice versa. Returns modified string. Only perform on text without html markup.

Returns: modified string

$string = String to be modfied
$to_ascii = true or false. False = from ascii code to standard. True = from standard to ascii code. Default = true.
